1. 속성 선택자

선택하고자 하는 요소내에 이미 기재되어있는 속성들을 활용해서 선택하는 방법
선택자 뒤에 []을 이용해서 속성과 속성값을 제시하면서 선택 

    선택자[속성=속성값]{
    선택자[속성~=속성값]{
    선택자[속성|=속성값]{
    선택자[속성^=속성값]{
    선택자[속성$=속성값]{
    선택자[속성*=속성값]{
        스타일속성:값;
    }

2. 자손선택자와 후손선택자

많은 요소들이 중첩되면서 작성될때
자손 : 바로 하위인 요소들 / 후손 : 하위 요소들 전부 

    * 자손선택자 : >
    a>b{
        스타일속성:값;
    }
    * 후손선택자 : 공백
    a b{
        스타일속성:값;
    }

3. 동위(같은레벨) 선택자

동위관계(같은레벨)에 있는 뒤에 위치한 특정 요소를 선택할 때 사용

    * a요소 바로 뒤에 있는 b요소 "하나만"을 선택
    a+b {
        스타일속성:값;
    }
    * a요소 뒤에 "모든" b요소 다 선택
    a~b{
        스타일속성:값;
    }

4. 반응 선택자

사용자의 움직임에 따라 적용되는 선택자

    * 특정 요소에 "클릭"되는 순간 스타일을 반영시키고싶을때
    선택자:active{
        스타일속성:값;
    }
    * 특정 요소에 "마우스가 올라간 순간" 스타일을 반영시키고싶을때
    선택자:hover{
        스타일속성:값;
    }

5. 상태 선택자

요소의 상태에 따라서 선택되는 선택자

    1) 체크된(checked) 상태의 요소에 스타일 부여하고 싶을때
       선택자:checked
    2) 초점(focus)이 맞춰진 input요소에 스타일 부여하고 싶을때
       선택자:focus
    3) 활성화/비활성화 된 요소에 스타일 부여하고 싶을때
       선택자:enabled
       선택자:disabled
