font-size : 글꼴의 크기를 변경할때 사용

선택자{font-size:글꼴크기(px|em|%);}
  • 일반글꼴
  • px테스트
  • em테스트
  • %테스트

font-weight: 글꼴의 굵기를 표현할 때 사용

    선택자{
        font-weight:normal|bold|bolder|lighter|100|400|900;
    }
  • 원래 굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게
