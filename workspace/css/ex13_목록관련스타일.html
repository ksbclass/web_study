목록 관련 스타일 (list~)

list-style-type : 불릿기호 변경시 사용

list-style-image : 불릿기호로 이미지를 적용시키고자 할때

list-style-postion : 불릿기호의 위치를 조정할때 사용

    * 순서없는 목록(ul)
    선택자 {
        list-style-type : disc|circle|square|none;
        list-style-image: url("이미지의 경로");
        list-style-position: inside|outside;
    }
    * 순서있는 목록(ol)
    선택자 {
        list-style-type : decimal|decimal-leading-zero;
               "        : lower-alpha|upper-alpha;
               "        : lower-roman|upper-roman;
    }
