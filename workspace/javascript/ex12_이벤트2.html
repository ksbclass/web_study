Key 이벤트

- keydown | keypress => 키가 눌려질때 발생되는 이벤트
> keydown : 키보드에서 키가 눌려질때 발생됨 . 모든키
> keypress : 키보드에서 키가 눌려져있는 상태. 펑션,기능키 등에서 발생 안됨.
- keyup => 키가 올라올때 발생되는 이벤트

태그별 기본적으로 가지고 있는 이벤트 제거

- a 태그 : 클릭시 href속성에 쓰여져 있는 url을 요청해주는 기본적인 이벤트존재
- form 태그의 submit 버튼 : submit 버튼을 클릭시, form태그의 action 속성에 쓰여져 있는 url로 입력값들과 함께 요청하는 기본 이벤트 존재
- 태그 내에 기본 설정된 이벤트를 실행을 막을때는 return false 로 설정함
