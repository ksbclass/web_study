배열관련 메소드

배열.concat(배열,배열,...)

배열.reverse() : 배열의 요소를 역순으로 변경

배열.sort() : 배열의 요소를 정렬허가

배열의 값을 추가, 조회 : push,pop

6-1) 배열.unshift(추가할 요소) : 배열의 맨 앞에 요소 추가 후 배열의 크기 반환
6-2) 배열.shift() : 배열의 맨 앞에 요소 제거 시키고 제거된요소 반환

7_1) 배열.slice(시작인덱스,끝인덱스) : 시작인덱스에서 끝 인덱스 까지 추출해서 반환 해주는 메소드
7_2) 배열.splice(시작인덱스,제거수,[추가 할 값]) : 배열의 요소 제거 및 추가
