브라우저의 언어임.
인터프리터 방색의 언어

자바스크립트의 특징

  • 접근제한자 개념이 없음.
  • 변수 선언이 필요가 없음. 초기화시 자료형 결정
  • 웹 환경에 특화된 언어 => 보안등의 이유로 기능이 제한됨
  • 보안에 취약함 : 소스가 외부 공개됨

자바스크립트의 사용법

  • 인라인 방식 (inline) : 태그내에 간단히 소스코드를추가
  • 내부(internal) : hteml 파일 내에 script 태그로 소스 코드 추가
  • 외부(external) : 별도의 .js 파일로 소스 코드를 추가.

인라인 방식

<태그 on 이벤트명 = "자바스크립트소스 코드 (해당 요소에 이벤트 발생시 실행시킬 소스코드)" >
<a href='javascript : 자바스크립트소스코드'>내용</a>
