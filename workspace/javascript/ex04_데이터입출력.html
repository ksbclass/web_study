데이터를 출력하는 구문

  • [window.]alert("브라우저의 확인창에 출력시킬 메시지")
  • [window.]console.log("브라우저의 콘솔창에 출력시킬 메시지")
  • document.write("브라우저의 화면에 출력시킬 메시지")
  • 태그.innerHTML("태그 내부의 출력시킬 메시지")

데이터를 입력하는 구문

  • 변수 = [window.]confirm("질문내용");
  • 변수 = [window.]prompt("질문내용");
  • 변수 = 특정요소.속성(id,className,innerHTML,innerText,...);

window.prompt("질문내용");

prompt 호출시 "질문내용"과 입력할 수 있는 "텍스트 상자"와 "확인/취소"버튼이 보여지는 브라우저의 알림창 발생
확인 버튼 클릭시 텍스트 상자에 입력된 value 값을 "문자열"로 변환, 취소 버튼 클릭시 null 반환 => 변수에 담아서 활용 가능
